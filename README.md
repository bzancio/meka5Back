# TypingMaster Web App

Aplicación web para practicar mecanografía con frases célebres y documentos subidos por el usuario, con análisis de errores y seguimiento del progreso.

## Funcionalidades

- 🔐 Login y gestión de usuarios
- ⌨️ Modo entrenamiento de mecanografía
  - Por tiempo
  - Por debilidades
  - Frases famosas
  - Documentos subidos por el usuario
- ⏱️ Pruebas cronometradas (15, 30, 60 segundos)
- ⚙️ Filtros de dificultad por caracteres
- 📊 Estadísticas de rendimiento (errores, velocidad, progreso)
